EIN Wallfahrtslied. Aus der Tiefe rufe ich, Herr, zu dir, / (a) Kla 3:55
höre auf meine Stimme! / Lass deine Ohren merken auf mein lautes Flehen! /
Wenn du die Sünden anrechnest, / Herr, wer kann bestehen? /
Doch bei dir ist Vergebung, / auf dass man dich fürchte. / (a) Jes 55:7
Ich hoffe auf dich, o Herr, / meine Seele hofft auf dein Wort. /
Meine Seele harrt auf den Herrn, / mehr als die Wächter auf den Morgen. / Mehr als die Wächter auf den Morgen, /
harre, Israel, auf den Herrn! / Denn bei dem Herrn ist die Gnade, / bei ihm ist reichlich Erlösung. /
Ja, er wird Israel erlösen / von all seinen Sünden.
